# iBizSim 智能决策辅助系统 — 设计构思

## 选定方案：「清新」— 延续 wuushuang.com 的明亮简洁风格

**设计运动**: 现代扁平化 + 功能色彩编码

**核心原则**:
1. 明亮留白 — 纯白/极浅灰背景，大量留白让内容呼吸
2. 色彩即功能 — 黄色=必填、橙色=选填、蓝色=公式、灰色=不填、绿色=正常、红色=超限
3. 卡片式组织 — 白色卡片 + 细边框 + 小圆角，层次分明
4. 温暖亲和 — 柔和的彩色气泡装饰，绿色主色调传递积极感

**色彩哲学**:
- 背景: 纯白 #ffffff + 极浅灰 #f9fafb
- 主强调色: 翠绿 #10b981 (Emerald-500)
- 次强调色: 温暖黄 #f59e0b (Amber-500)
- 文字: 深灰 #111827 (Gray-900)
- 辅助文字: #6b7280 (Gray-500)
- 边框: #e5e7eb (Gray-200)
- Hero 装饰: 柔和的多彩气泡（粉、绿、黄、蓝）

**布局范式**:
- 仪表盘布局：左侧固定侧边栏(240px) + 右侧主内容区
- 侧边栏：白色背景，分组导航，图标+文字
- 主内容区：卡片式布局，折叠面板展示详细数据
- 顶栏：简洁，含面包屑和用户信息

**签名元素**:
- 彩色气泡装饰 — Hero 区域和空状态页面使用柔和的渐变气泡
- 颜色编码输入框 — 延续 wuushuang.com 的黄/橙/蓝/灰色编码系统

**交互哲学**: 即时反馈，简洁直观。折叠面板展开/收起，表格内联编辑。

**动画**: 轻柔的过渡（200ms），折叠面板的平滑展开，按钮的微妙悬停效果。

**字体系统**: 系统中文无衬线字体（-apple-system, "PingFang SC", "Microsoft YaHei"），数据用等宽字体。
