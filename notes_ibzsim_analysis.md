# iBizSim 竞赛规则分析笔记

## 一、竞赛流程（来自决策向导图 + 操作指南）

1. 查看公共报表（市场份额、分项指标排序）
2. 制定战略
3. 四部门并行决策：
   - **市场部**：查看公共报表的市场份额和市场价格 → 制定市场营销策略
   - **生产部**：查看内部报表的期末产品状况和期末企业状况 → 制定生产安排决策
   - **人事部**：查看内部报表的期末企业状况 → 制定人力资源决策
   - **财务部**：查看内部报表的会计项目和期末净资产和期末企业状况 → 制定融资投资策略
4. 提交决策
5. 查看模拟结果 → 循环回到步骤1

## 二、决策域完整清单（来自快捷模式决策表单）

根据操作指南1.5.5，决策表单包含以下域：
- 销售计划（定价、广告费、促销费）× 4产品 × 4市场
- 运输计划（各产品运往各市场的数量）
- 生产计划（各产品各班次的产量、原材料订购）
- 人力与原材料（雇佣/解聘工人、工资系数）
- 财务（贷款、国债、债券、分红）

## 三、各决策域详细参数

### 3.1 市场/销售决策
- 4种产品 × 4个市场 = 16个定价
- 4种产品的广告费
- 4个市场的促销费
- 4种产品 × 4个市场 = 16个运输量

### 3.2 生产决策（当前已实现）
- 4种产品 × 4班次 = 16个产量格
- 雇佣/解聘工人
- 购买机器
- 原材料订购量

### 3.3 研发决策
- 4种产品的研发费用

### 3.4 人力/员工决策
- 工资系数
- 雇佣工人数
- 解聘工人数（含退休）

### 3.5 财务决策
- 银行贷款
- 发行债券
- 购买国债
- 分红

## 四、关键规则参数

### 生产资源
- 产品A: 机器121h, 人力68h, 原材料690单位
- 产品B: 机器201h, 人力109h, 原材料1090单位
- 产品C: 机器259h, 人力121h, 原材料1900单位
- 产品D: 机器372h, 人力148h, 原材料3060单位
- 正班520h/期, 加班260h/期

### 人力规则
- 新工人培训费2000元/人，培训期效率25%
- 每期3%退休
- 解聘+退休 ≤ 期初工人10%
- 招聘 ≤ 期初工人50%
- 退休/解聘安置费3000元/人

### 机器规则
- 价格120,000元，折旧5年（每期5%）
- 购买后下期运输安装，再一期使用
- 一加班+二班机器总数 ≤ 公司机器总数

### 财务规则
- 初始现金6,000,000元
- 贷款总额上限12,000,000元，年利率6%
- 国债年利率3%
- 债券年利率9%，每期偿还5%本金
- 税率20%

### 评判标准（7项指标权重）
- 本期利润: 0.15
- 市场份额: 0.16
- 累计分红: 0.15
- 累计缴税: 0.12
- 净资产: 0.16
- 人均利润率: 0.13
- 资本利润率: 0.13

## 五、决策域之间的数据依赖

- 生产决策 → 需要知道工人数（人力决策）、机器数、原材料
- 销售决策 → 需要知道生产量（生产决策）来决定运输量
- 财务决策 → 需要知道所有成本（生产、人力、运输、广告等）来决定贷款/分红
- 研发决策 → 影响产品等级 → 影响定价和需求
- 工资系数 → 影响产品等级和废品率

## 六、关键发现

1. **决策是每期一次性提交**：所有域在同一个决策表单中，一次提交
2. **四部门并行但数据相互依赖**：生产需要人力，销售需要产量，财务需要总成本
3. **当前 rule.xls 的参数与官方规则不同**：
   - rule.xls: A(110,80,500), B(150,100,800), C(180,110,1600), D(280,140,2500)
   - 官方规则: A(121,68,690), B(201,109,1090), C(259,121,1900), D(372,148,3060)
   - 说明不同比赛场景参数不同，系统需要支持参数可配置
